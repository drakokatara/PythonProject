5. Προετοιμασία Περιβάλλοντος & Εγκατάσταση (Setup & Deployment)

Η εφαρμογή βασίζεται στο Django και η διαδικασία εγκατάστασης έχει απλοποιηθεί ώστε να μπορεί να εκτελεστεί γρήγορα σε οποιοδήποτε τοπικό περιβάλλον ανάπτυξης.

5.1 Λήψη του Κώδικα (Clone από GitHub)

Το πρώτο βήμα είναι η λήψη του κώδικα στον τοπικό σας υπολογιστή. Ανοίξτε το τερματικό (Command Prompt ή Terminal) και πληκτρολογήστε:

Bash

git clone https://github.com/your-username/pcs-attendance.git
cd pcs-attendance


(Σημείωση: Αντικαταστήστε το link με το πραγματικό URL του repository σας).

5.2 Προετοιμασία Virtual Environment

Για την απομόνωση των εξαρτήσεων και την αποφυγή συγκρούσεων με άλλες εγκαταστάσεις Python:

Δημιουργία περιβάλλοντος: python -m venv venv

Ενεργοποίηση (Windows): venv\Scripts\activate

Ενεργοποίηση (Linux / Mac): source venv/bin/activate

5.3 Εγκατάσταση Απαραίτητων Πακέτων

Η εφαρμογή χρησιμοποιεί το pip για την εγκατάσταση του Django και των εξειδικευμένων βιβλιοθηκών για αργίες και ημερομηνίες:



Bash

pip install django holidays python-dateutil


Django: Το κεντρικό framework της εφαρμογής.

Holidays: Για τον αυτόματο εντοπισμό των επίσημων αργιών στην Ελλάδα.

Python-dateutil: Για τον ακριβή υπολογισμό των μηνιαίων περιόδων.

5.4 Ρύθμιση Βάσης Δεδομένων (Migrations)

Μετά την εγκατάσταση, πρέπει να δημιουργηθεί η δομή της βάσης δεδομένων (SQLite) που θα φιλοξενήσει τα μοντέλα Employee και Attendance:



Bash

python manage.py makemigrations
python manage.py migrate


Αυτή η διαδικασία δημιουργεί αυτόματα τους πίνακες για τους εργαζόμενους, τις παρουσίες και το αρχικό χρέος.

5.5 Εκτέλεση της Εφαρμογής

Ξεκινήστε τον τοπικό server ανάπτυξης με την εντολή:

python manage.py runserver

Μετά την εκτέλεση, ανοίξτε τον browser σας και επισκεφθείτε τη διεύθυνση:

http://127.0.0.1:8000/

5.6 Συνοπτικός Έλεγχος Setup

Ακολουθώντας τα παραπάνω βήματα, ο χρήστης εξασφαλίζει:

Καθαρό περιβάλλον: Μέσω του virtual environment.

Συγχρονισμό: Με την τελευταία έκδοση κώδικα από το GitHub.

Λειτουργικότητα: Με την αρχικοποίηση της βάσης δεδομένων και την πρόσβαση στο Dashboard.
